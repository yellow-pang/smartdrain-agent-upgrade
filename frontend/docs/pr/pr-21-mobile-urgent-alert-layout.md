## PR 제목

[fix] 모바일 긴급 알림 패널 잘림 수정

## 작업 내용

모바일 기기 폭에서 긴급 알림 패널의 왼쪽 제목과 설명이 화면 밖으로 밀리는 레이아웃 문제를 수정했습니다.

- 모바일 기본 화면에서는 긴급 알림 패널을 뷰포트 기준 fixed 위치로 표시합니다.
- 패널 좌우에 `12px` 여백을 적용해 360px 폭에서도 화면 안에 표시되도록 했습니다.
- `sm` 이상 화면에서는 기존처럼 알림 버튼 아래 오른쪽에 표시되는 absolute 팝오버 동작을 유지합니다.
- 알림 목록, 빈 상태, 모두 읽음, 시설 상세 링크 이동의 기존 동작은 변경하지 않았습니다.

| `docs/plans/plan-16-mobile-urgent-alert-layout.md` | 모바일 알림 패널 문제 원인과 대응 방향 기록                  |

## 변경 전/후

| 항목               | 변경 전                                         | 변경 후                                  |
| ------------------ | ----------------------------------------------- | ---------------------------------------- |
| 모바일 패널 기준점 | 알림 버튼의 오른쪽 끝                           | 뷰포트의 좌우 여백                       |
| 360px 폭 표시      | 패널 왼쪽이 화면 밖으로 밀려 텍스트 일부가 잘림 | 패널 전체와 제목·설명이 화면 안에 표시됨 |

## 리뷰 포인트

- 모바일 fixed 패널은 헤더 아래에 표시되며, 기존 `z-50`을 유지합니다.
- 여러 알림이 있을 때는 기존 내부 목록 스크롤을 그대로 사용합니다.
- 실제 모바일 브라우저의 동적 주소 표시줄 변화에서도 패널 위치가 자연스러운지 확인이 필요합니다.
