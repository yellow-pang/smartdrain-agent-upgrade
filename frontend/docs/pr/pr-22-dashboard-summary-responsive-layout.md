## PR 제목

[fix] 대시보드 현황 반응형 우선순위 개선

## 작업 내용

대시보드 현황 카드를 단순히 같은 크기로 나열하지 않고, 관제에서 먼저 판단해야 할 전체 현황과 위험 상태가 더 잘 보이도록 반응형 배치를 조정했습니다.

- 모바일에서는 `전체` 카드가 한 행 전체를 사용합니다.
- 태블릿에서는 `전체` 1열과 `위험` 2열을 첫 행에 배치합니다.
- 데스크톱에서는 위험 카드만 다른 상태 카드보다 넓게 표시합니다.
- 실제 데이터와 로딩 스켈레톤에 같은 grid 규칙을 적용했습니다.

## 리뷰 포인트

- 상태 데이터 순서는 전체, 위험, 주의, 양호, 판단불가를 유지합니다.
- 위험 색상(red)뿐 아니라 카드 폭으로도 우선순위를 드러냅니다.
- 640px 이상의 중간 화면에서 첫 행이 `전체` 1열 + `위험` 2열로 의도대로 배치되는지 확인이 필요합니다.
