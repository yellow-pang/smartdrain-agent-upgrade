## PR 제목

[fix] 긴급 알림 패널 웹 표시 개선

## 작업 내용

- 웹 알림 패널 폭을 넓혀 `모두 읽음`, 시설명, 상태 배지의 불필요한 줄바꿈을 줄였습니다.
- 알림 목록 높이를 늘리고 스크롤 영역을 확장해 4개 이상의 알림도 확인하기 쉽게 했습니다.
- 알림을 `updatedAt` 기준 최신순으로 표시하도록 정렬했습니다.
- 같은 시설 알림이 갱신될 때 기존 위치에 남지 않고 최신 알림으로 맨 위에 올라오도록 store upsert 순서를 조정했습니다.
- 시설명은 한 줄 말줄임, 상태 배지는 고정, 메시지는 최대 2줄로 제한해 웹에서 스캔하기 좋게 정리했습니다.
- `새 알림`/`읽은 알림` 탭을 추가하고, 읽은 알림은 최근 10개까지 확인할 수 있게 했습니다.
- 알림 클릭 또는 `모두 읽음` 클릭 시 새 알림을 제거하지 않고 읽은 알림 탭으로 옮기도록 변경했습니다.

## 리뷰 포인트

- 웹 화면에서 `모두 읽음` 버튼이 줄바꿈되지 않는지 확인합니다.
- 긴 시설명과 상태 배지가 한 줄 안에서 자연스럽게 표시되는지 확인합니다.
- 알림 4개 이상이 들어왔을 때 스크롤로 확인 가능한지 확인합니다.
- 동시에 여러 알림이 들어오거나 같은 시설이 다시 갱신될 때 최신 알림이 위에 표시되는지 확인합니다.
- 알림 클릭 후 상세 페이지로 이동하면 해당 알림이 읽은 알림 탭으로 이동하는지 확인합니다.
- `모두 읽음` 클릭 시 새 알림 목록이 비고 읽은 알림 탭에 최근 10개까지 보관되는지 확인합니다.
