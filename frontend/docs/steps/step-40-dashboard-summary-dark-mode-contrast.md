# 40 대시보드 현황 다크 모드 대비 보정 결과

## 작업 결과

다크 모드의 대시보드 현황 카드에서 `전체` 수치가 `text-slate-900`으로 고정되어 어두운 카드 배경에서 잘 보이지 않던 문제를 수정했다.

상태별 수치는 기존 위험도 의미를 유지하면서, 다크 모드에서만 더 밝은 색상 단계로 표시하도록 보정했다.

| 항목 | 라이트 모드 | 다크 모드 | 목적 |
| --- | --- | --- | --- |
| 전체 | `slate-900` | `slate-100` | 배경 대비를 확보하고 일반 집계값으로 유지 |
| 위험 | `red-600` | `red-400` | 위험 상태의 red 의미를 유지하면서 가독성 보강 |
| 주의 | `amber-600` | `amber-400` | 주의 상태의 amber 의미를 유지하면서 가독성 보강 |
| 양호 | `emerald-600` | `emerald-400` | 양호 상태의 green 의미를 유지하면서 가독성 보강 |
| 판단불가 | `slate-500` | `slate-300` | 중립 상태를 유지하면서 배경과의 명도 차 확보 |

색상 계열을 바꾸지 않아 위험·주의·양호·판단불가의 기존 상태 의미와 충돌하지 않는다. `전체`만 중립 고대비 색상을 사용하므로 상태값으로 오해되지 않는다.
